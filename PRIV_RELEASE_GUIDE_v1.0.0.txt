## 🔧 TROUBLESHOOTING

### Problem: Tag tidak terdeteksi Packagist
**Solusi:**
- Pastikan webhook GitHub aktif di Packagist settings
- Manual trigger update via API
- Cek composer.json valid dengan: `composer validate`

### Problem: Version conflict
**Solusi:**
- Hapus tag jika masih lokal: `git tag -d v1.0.0`
- Fix versi di composer.json
- Buat tag ulang

### Problem: File deprecated masih masuk package
**Solusi:**
- Cek composer.json section `archive.exclude`
- Cek .gitattributes export-ignore
- Buat tag baru setelah fix
